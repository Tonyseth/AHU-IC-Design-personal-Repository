## 📚 简介

这是一个致力于分享安徽大学集成电路专业学习资料的开源仓库。我们的目标是为所有学习者提供一个公平、开放的学习平台。

### 🎯 我们的使命

- 🚫 杜绝学习资料的商业化和非法牟利
- 🔍 为寻找资料的同学提供便利
- 🏁 确保所有学习者站在同一起跑线上公平竞争

### 🤝 您可以做什么

- ✅ 自由使用和下载仓库内容
- 📤 上传并完善仓库（注意：不接受安大老师的PPT）
- 🖊️ 修正代码和笔记中的错误
- 🆓 免费、无条件地与其他同学分享
